# Setup and: injectivity of L implies equivariant log-concavity

## 1. The ring R

Fix a matroid `M` on `[n] = {1,...,n}`, with circuit set `𝒞(M)`. Define

  `R = k[x_1, ..., x_n] / ( x_I : I ∈ 𝒞(M) ) + (x_i^2 : i ∈ [n]).`

(The added `x_i^2` makes the dimension claim hold; without it, R is the Stanley–Reisner
ring of the independence complex `IN(M)` and has Krull dimension `rk(M)`, not artinian.)

Equivalently `R` is the quotient of the *squarefree polynomial algebra* by the
monomial ideal generated by `{x_I : I ⊆ [n] dependent}` — note that `{I dependent}`
is closed under supersets, so it suffices to use circuits.

The image of `x_S = ∏_{i∈S} x_i` is nonzero iff `S` is independent. The set
`{x_S : S ∈ Indep(M)}` is a `k`-basis. So

  `dim_k R_d = f_d(M) = #{S ∈ Indep(M) : |S| = d}`, with `Hilb(R,t) = ∑ f_d t^d.`

The group `Aut(M)` acts on `R` by permuting the `x_i`'s, and `R_d` is the
permutation representation `k[Indep_d(M)]`.

**Remark.** `R` is exactly the "matroid face ring mod squares" or "exterior face ring
of `IN(M)`, made commutative". Stanley considered the SR ring of `IN(M)` (Krull dim
`= rk(M)`) — our `R` is the artinian reduction by `(x_i^2)` (which is *not* a regular
sequence in general; for matroids of rank `≥ 2` the squares already vanish on top, and
the regular system of parameters is more subtle). The face-ring literature usually
takes the artinian reduction by `n - rk(M)` linear forms (giving `h`-vector in degree),
not by squares.


## 2. The dual R^∨ and S = R ⊗ R^∨

Let `R^∨ = Hom_k^{gr}(R, k)`, the graded linear dual concentrated in non-positive
degrees, with `(R^∨)_{-d} = (R_d)^*`. Pick the basis dual to the monomial basis:
`y_S ∈ R^∨_{-|S|}` with `⟨y_S, x_T⟩ = δ_{S,T}` for `S, T` independent.

Make `R^∨` into a left `R`-module by transpose of multiplication:
  `(x_i · φ)(r) := φ(x_i · r)`, so `x_i · y_S = y_{S\{i}}` if `i ∈ S`, else `0`.

Concretely: `x_i` acts on `R^∨` by **contraction** — it is essentially `∂_{x_i}`
seen on the dual basis.

Let `S := R ⊗_k R^∨`. As a graded vector space,

  `S_{a-b} = R_a ⊗ R^∨_{-b}`, and `dim S_d = ∑_{a-b=d} f_a · f_b`.

I will use two gradings:
- **external degree** `e = |S| - |T|` for `x_S ⊗ y_T`
- **internal degree** `m = |S|`  (so `|T| = m - e`)

Then `(R_m ⊗ R^∨_{-(m-e)})` is the `(m,e)`-bigraded piece, dim `= f_m · f_{m-e}`.


## 3. The Lefschetz operator L

  `L = ∑_{i=1}^n  x_i ⊗ x_i  ∈ End_k(S)`

where the first `x_i` acts by multiplication on the left factor and the second `x_i`
acts on the right factor by contraction (as defined above). Explicitly,

  `L(x_S ⊗ y_T) = ∑_{i ∈ T, i ∉ S, S ∪ {i} ∈ Indep(M)}  x_{S ∪ {i}} ⊗ y_{T \ {i}}.`

On bigraded pieces, `L : (m, e) → (m+1, e+2)`. It is `Aut(M)`-equivariant since the
sum is over all `i` and each summand is symmetric in `i`'s label.

**Higher powers.** For each `d ≥ 1`, expanding `L^d`:

  `L^d(x_S ⊗ y_T) = ∑_{(i_1,...,i_d)}  x_{S ∪ {i_1,...,i_d}} ⊗ y_{T \ {i_1,...,i_d}}`

summed over ordered `d`-tuples of distinct elements of `T \ S` such that each prefix
`S ∪ {i_1, ..., i_k}` is independent. Since independence is closed under subsets,
the prefix condition reduces to **`S ∪ {i_1, ..., i_d}` independent**. Counting orderings:

  `L^d(x_S ⊗ y_T) = d! · ∑_{T' ⊆ T : |T'| = d,  S ⊔ T' ∈ Indep(M)}  x_{S ⊔ T'} ⊗ y_{T \ T'}.`

In particular, taking `S = ∅` and `|T| = d`:

  `L^d(1 ⊗ y_T) = d! · x_T ⊗ 1`  (since `T` independent ⇒ every subset is).

So `L^d` is at least *injective* on the "off-diagonal corner" `m = 0` — the `Aut(M)`-equivariant
map `1 ⊗ y_T ↦ d! · x_T ⊗ 1` is multiplication by `d!`, hence iso on this piece.
The interesting question is what happens on `m > 0` pieces.


## 4. Injectivity of L implies equivariant log-concavity

**Claim.** If `L : S_{-d} → S_{-d+2}` is injective for some `d ≥ 2`, then for every `m ≥ 0`,

  `[f_{m+1}]^2 - [f_m] · [f_{m+2}] ≥ 0  in  Rep(Aut(M))`,        (*)

where `[f_k] := k[Indep_k(M)] ∈ Rep(Aut(M))` and `·` is internal tensor.

Actually, let me state it more carefully: I want ELC on `(f_m, f_{m+1}, f_{m+2})`.

**Proof (case d=2, the basic case).** Decompose into internal-degree pieces.
`L` preserves internal degree only up to a shift `m ↦ m+1`, so consider the *direct sum*

  `L : ⊕_m (R_m ⊗ R^∨_{-(m+2)}) → ⊕_m (R_{m+1} ⊗ R^∨_{-(m+1)})`,

which is `L : S_{-2} → S_0` summed over bigraded components. The block from `(m, -2)`
goes to `(m+1, 0)`. **The blocks are independent** — `L` increases `m` by exactly `1`
and is injective on `S_{-2}` iff it is injective on each `(R_m ⊗ R^∨_{-(m+2)})`.

So injectivity of `L : S_{-2} → S_0` gives, for each `m`,

  `L : R_m ⊗ R^∨_{-(m+2)} ↪ R_{m+1} ⊗ R^∨_{-(m+1)}`

between `Aut(M)`-representations. As `Aut(M)`-reps,
- `R_m ⊗ R^∨_{-(m+2)}` is naturally `[f_m] · [f_{m+2}]^*` and, since `[f_{m+2}]` is a
  permutation rep on `Indep_{m+2}`, it is *self-dual*. So this is `[f_m] · [f_{m+2}]`.
- `R_{m+1} ⊗ R^∨_{-(m+1)} = [f_{m+1}] · [f_{m+1}]^* = [f_{m+1}]^2`.

The cokernel `[f_{m+1}]^2 − [f_m] [f_{m+2}]` is an honest representation (the cokernel
of an injective map of finite-dimensional reps). This is precisely (*).               ∎

**General d.** The same argument applied to `L^{d-1} : S_{-d} → S_{d-2}` (degree `2(d-1)`)
gives, on the `m`-block,

  `L^{d-1} : R_m ⊗ R^∨_{-(m+d)} ↪ R_{m+d-1} ⊗ R^∨_{-(m+1)}`

i.e. `[f_m] [f_{m+d}] ↪ [f_{m+d-1}] [f_{m+1}]`. As inequalities of *Schur-type* products:

  `[f_{m+1}] [f_{m+d-1}] ≥ [f_m] [f_{m+d}]  in Rep(Aut(M))`.

This is the "Newton" form of log-concavity — equivalent in this generality to the standard
ELC on `(f_k)` (you recover ELC at `k` by setting `d = 2`, `m ↦ k-1`).

**Why injectivity is special to permutation reps.** Self-duality of `[f_k]` is essential
for the right-hand side to be a "square". If `Aut(M)` acted non-trivially on `Indep_k`
(beyond permutations) one would need to track duals more carefully. As written this works
verbatim for the standard `Aut(M)`-action.


## 5. Note on `Aut(M)`-equivariance

`L = ∑_i x_i ⊗ x_i` is the *diagonal* of the tensor `(∑ x_i) ⊗ (∑ x_i)` only up to
the off-diagonal terms `x_i ⊗ x_j` for `i ≠ j`. Why is `L`, not `(∑ x_i) ⊗ (∑ x_i)`,
`Aut(M)`-equivariant? Because `Aut(M)` is finite and permutes the basis `{e_1,...,e_n}`
of `k^n`: the diagonal tensor `∑ e_i ⊗ e_i` is permutation-invariant whereas the
non-diagonal terms are not. So `L` corresponds to the Aut(M)-invariant "diagonal class"
in `k^n ⊗ k^n` — analogous to the *Casimir* in a Lie-algebra context.

This is reminiscent of the **Casimir-type Lefschetz** used by Soergel/BHMPW in the
intersection-cohomology module of a matroid.
