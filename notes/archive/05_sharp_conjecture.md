# The sharp conjecture: Q must be full-rank

## What the PSD-Q experiment showed

For `L_Q = ∑_{i,j} q_{ij} x_i ⊗ x_j` (operator on `S = R ⊗ R^∨`, multiplication-by-`x_i`
on left factor, contraction-by-`x_j` on right factor), tested on M(K_4) and Fano:

| `Q` shape (n × n)             | rk(Q)  | L_Q^d iso d≥2? |
|--------------------------------|--------|----------------|
| `I_n` (identity)              | n      | **ISO**        |
| `M^T M` random (generic)      | n      | **ISO**        |
| `M + M^T` random sym          | n (gen.)| **ISO**       |
| `uu^T + vv^T`                 | 2      | NOT-ISO        |
| `uu^T`                        | 1      | NOT-ISO        |

**Conclusion.** Hard Lefschetz holds iff `Q` has full rank `n`. Positivity is not required;
only invertibility/non-degeneracy.

## Sharper conjecture

  **SHARP CONJ-A.** For every matroid `M`, every `d ≥ 2`, and every invertible
  `Q ∈ GL_n(k)`,

      `L_Q^d : S_{-d}  →  S_d`   is an isomorphism.

  In particular, hard Lefschetz holds for a Zariski-open dense set of "Lefschetz
  bilinear forms".

## Why this is the right framing

### Kraus / CP-map perspective

When `Q = ∑_α v_α v_α^T` (SVD/spectral decomp), `L_Q` corresponds to a CP map on
`End(R)` with Kraus operators `K_α = M_{v_α \cdot x} = M_{∑ v_α^i x_i}`.

- If `rk(Q) = r`, we have `r` Kraus operators (linear forms in `R_1`).
- The algebra generated by these `r` operators is `k[v_1·x, ..., v_r·x]` modulo the
  matroid relations.
- For `r < n`, this subalgebra is **strictly smaller** than `R`.

So:

  **GENERAL PRINCIPLE.** `L_Q^d` is iso `S_{-d} → S_d` iff the Kraus operators
  generate `R` as a `k`-algebra ⟺ `rk(Q) = n` ⟺ `Q ∈ GL_n`.

This is **completely analogous to Lefschetz for Frobenius algebras**: an element
acts non-degenerately iff it is in the "regular" part.

### Bilinear-form / cohomological perspective

`Q` defines a bilinear form on `k^n = R_1`. The form is non-degenerate iff `Q ∈ GL_n`.
Via the natural pairing `R_1 ⊗ R_1^* → k`, we get `Q ∈ R_1 ⊗ R_1`. The user's
`L_Q` is the **Lefschetz operator induced by `Q`**.

In the AHK Chow-ring story, the Lefschetz element is a single class `ℓ ∈ A^1`,
which is a *generic* linear form. In our story, the Lefschetz is a **bilinear form**
(a rank-2 tensor), and the condition is non-degeneracy of this form.

  This is **more analogous to Hodge-Riemann** than to Hard Lefschetz: HR uses a
  *bilinear form* (the polarization), and the relevant condition is non-degeneracy
  / positivity.

So we are doing "Hard Lefschetz / Hodge-Riemann on `End(R)`" with the polarization
being `Q`.

### Why not check positivity?

The data above shows even non-PSD invertible `Q` (e.g., `M + M^T` for random `M`)
works. So the operative condition is **rank-n**, not positivity.

But Hodge-Riemann *signature* (needed e.g. for log-concavity of *real* sequences) is
about positivity. For ELC, we only need *rank* (injectivity gives the bare
dim inequality, with virtual rep = effective rep).

  **REMARK.** If `Q` is *positive-definite real symmetric*, we expect a
  **Hodge-Riemann-like statement** on `S`: a primitive bilinear form of fixed
  signature. This would not just give ELC, but **strict ULC** (ultra-log-concavity)
  — a stronger statement.

## How to prove SHARP CONJ-A (sketches)

Three strategies that all generalize from `L = L_I`:

### 1. Direct sum + deletion-contraction induction

`S(M ⊕ M') = S(M) ⊗ S(M')`, with `L_Q(M⊕M') = L_Q ⊗ id + id ⊗ L_Q`. By Künneth-Lefschetz,
hard Lefschetz on factors gives hard Lefschetz on the sum. So WLOG `M` is connected.

For connected `M`, deletion-contraction gives short exact sequence
  `0 → S(M/e)[shift] → S(M) → S(M\e) → 0`?
(Not quite — the directions of functoriality conflict. Needs care.)

Modulo getting the deletion-contraction sequence right, induction on `n` proves
the result.

### 2. Spectral analysis via CP-map theory

`T_{L_Q}` on `End(R)` is, in the Kraus-form picture, a sum of *n* completely positive
rank-1 channels (when `Q` is diagonal). Its spectrum on each Aut(M)-isotypic
component can be computed via standard CP-map techniques (Lindblad-like).
Hard Lefschetz ⟺ each isotypic spectrum has the right zero/non-zero structure.

This connects to **Brion's spectral theory of equivariant cohomology**.

### 3. Direct combinatorial argument via "transfer matrices"

The matrix entries of `L_Q^d` (in monomial basis) are explicit polynomial expressions
in `q_{ij}`. Generically (for `Q ∈ GL_n`), the rank of `L_Q^d` is the maximum it
can be, which by Künneth is full rank.

This is the most direct approach but requires careful matroid-combinatorial bookkeeping.

## What SHARP CONJ-A implies that the original doesn't

If `L_Q^d` is iso for *every* `Q ∈ GL_n`, then we have a 2-parameter family of
Lefschetz elements. Differentiating in `Q` gives **secondary** Lefschetz operators
(higher Lie-theoretic invariants). These would give finer log-concavity statements
(maybe "logconcavity of moment-generating-functions of the f-vector").

## Bottom line

The conjecture is **better stated as**:

> *For every matroid `M` and every invertible `Q ∈ GL_n(k)`, the operator
> `L_Q^d : S_{-d} → S_d` is `Aut(M, Q)`-equivariantly an isomorphism for `d ≥ 2`.*

Here `Aut(M, Q) ⊆ Aut(M)` is the stabilizer of `Q` (for the standard `Q = I`, this is
all of `Aut(M)`).
